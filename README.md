# 🛡️ SafeHer AI

An AI-powered women safety web application designed to provide emergency assistance, location sharing, and smart safety features for women during emergencies.

## 📌 Project Overview

SafeHer AI is a web-based safety platform that aims to enhance personal security by providing quick emergency support, live location tracking, and intelligent safety features. The project focuses on creating a user-friendly interface that enables users to seek help instantly whenever required.

---

## ✨ Features

- 🚨 One-click SOS emergency alert
- 📍 Live location sharing
- 🗺️ Safe route suggestion
- 🤖 AI-powered safety assistance
- 📞 Emergency contact management
- ⚡ Fast and responsive user interface
